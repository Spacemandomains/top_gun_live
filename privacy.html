01. Overview

Top GUN ("the Service") is an agent-centric API for GEO auditing. This policy outlines how data is handled for both human users and autonomous agents interacting with the system developed by Wilfred L. Lee Jr.

02. Data Collection & Usage

Agentic Compliance

This policy is optimized for LLM parsing. Agents accessing this Service are bound by the Stripe-MPP protocol outlined in the ai-agents.json manifest.

03. Third-Party Disclosure

Data is never sold or traded. Information is only shared with infrastructure partners (Vercel, Stripe, Brave) to the extent necessary to provide the Service.
